## Docker Setup
> Hello World 출력<br />
`$ docker container run ubuntu:latest /bin/echo 'Hello World!'`<br />

> Docker Version 확인<br />
`$ docker version`<br />

> Docker 실행 환경 확인<br />
`$ docker system info`<br />

> Docker Disk 이용 상황 확인<br />
`$ docker system df`<br />
`$ docker system df -v`
